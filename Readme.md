# Quick Dodge (Android MVP)

Мини-игра для быстрого релиза в GetApps: игрок уклоняется от падающих блоков, набирает очки и пытается побить рекорд.

## Что уже реализовано

- Один экран, быстрый цикл игры на `Canvas`
- Управление пальцем (влево/вправо)
- Счёт и сохранение лучшего результата
- Экран старта и перезапуска
- Портретный режим, fullscreen

## Как запустить

1. Установи Android Studio (или Android SDK + эмулятор).
2. Открой проект.
3. Выполни сборку debug:

```bash
./gradlew :app:assembleDebug
```

## Как быстро монетизировать

Для первой версии (быстрый запуск):

1. **Реклама после проигрыша** (interstitial каждые 2-3 рана).
2. **Rewarded-реклама**: "Вторая жизнь" за просмотр.
3. **Покупка без рекламы** как единственный IAP.

Рекомендуемый поток:

- День 1: MVP (этот проект) + события аналитики.
- День 2: SDK рекламы + баланс частоты показа.
- День 3: иконка, скриншоты, описание, публикация.

## Публикация в GetApps (чеклист)

1. Подними `versionCode` и `versionName`.
2. Подпиши release build своим keystore.
3. Подготовь:
   - иконку 512x512
   - 3-5 скриншотов
   - короткое и полное описание
   - политику конфиденциальности (URL)
4. Загрузи `.aab`/`.apk` в консоль GetApps.
5. После модерации включи промо-акции и A/B по креативам.

## Идеи для удержания (быстрые)

- Ежедневная награда (coins)
- 3 скина игрока за просмотр рекламы
- Мини-миссии: "Выжить 30 секунд"
