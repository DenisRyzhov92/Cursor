# GetApps Listing (ru-RU)

## App Name
Космоферма: BioGel Империя

## Short Description
Развивай космоферму, запускай дронов и терраформируй новую планету.

## Full Description
Создай свою космоферму с нуля и преврати маленький купол в полноценную терраформинг-империю.

Твоя задача проста:
- собирай BioGel вручную,
- вкладывай ресурсы в апгрейды,
- автоматизируй добычу дронами,
- ускоряй прогресс и открывай новые технологии.

### Что внутри
- Idle clicker с плавным ростом дохода.
- Апгрейды для тапа, пассивной добычи и глобального множителя.
- Оффлайн-прогресс: ферма работает, пока тебя нет.
- Простая и понятная экономика без лишнего перегруза.
- Космическая тематика: купола, дроны, орбитальные модули, терраформинг.

### Rewarded-only монетизация
- Автоматической рекламы нет.
- Награда за рекламу дается только по кнопке игрока.
- Можно получить ускорение добычи и бонусный BioGel.

Построй эффективную орбитальную агросистему, оптимизируй цепочки добычи и выведи свою планету на новый уровень жизни.
